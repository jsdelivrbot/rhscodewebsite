About the RHS Code Club

The RHS Code Club is a club that meets every Wednesday to chill, code cool stuff, and most certainly not fool around in a computer lab (no way). Each week, you will learn new code and apply it in creative ways to your own coding creations, such as personal websites, online games, and other fun, cool, and hilarious projects.

FAQ's

Q: I don't know any coding at all, can I still join?

A: There is so much code out there, and as such, the RHS Code Club not only accepts members with litte to no coding knowledge, but teaches a lot of code that even advanced coders may have looked over, so you will be just as knowledgable as anyone else.
